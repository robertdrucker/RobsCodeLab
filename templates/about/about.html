About

Site Purpose

Rob's Code Lab serves as a personal journal for keeping track of lessons learned about web development and computational problem solving. It also functions as a virtual laboratory where selected code snippets can be methodically reviewed, investigated, and analyzed. Thus, this is a place to reflect upon code, ingrain key programming concepts, and become a better developer.

For All

Although this site functions as a personal code lab, I made it public to encourage information exchange with other coders. I find that holding discussions with other web developers is one of best ways to enhance programming skills.

Features

At the heart of this website is a full-feature blog and comment system, and there are individual blogs devoted to developing with CSS, HTML5, JavaScript, PHP, Python, and SQL. Each of these is a core language or scripting tool I regularly use for building web applications.

Since Rob's Code Lab is new, it does not yet have a lot of content. However, I will be adding new coding notes regularly, and additional features are planned. These include course reviews, book reviews, and expanded coverage of web-app languages, libraries, and frameworks.

A Short Bio

Hi! I am Robert Drucker, the designer and developer of this site. After working in the chemical industry as a process engineer and manager for several years, I developed a love for building websites. I have built web applications with various programming languages, libraries, and frameworks. These include PHP, Python/Flask, JavaScript, React, and NodeJS.

Professionally, my definite chief aim is to work with a team to build better websites. I also strive to keep learning and improving. I am from Louisville, Kentucky and currently reside with my family in Charleston, South Carolina.
