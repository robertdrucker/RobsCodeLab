Summary

A decorator can be used in Python to enhance the functionality of a function without changing the function. The code in this post utilizes a decorator to enhance or “prettify” a function that compares the output of three different trigonometric functions across a given set of values.

The example given here also provides a good illustrtion of how the unpacking operator * is used. This operator allows a variable number of positional arguments to be passed into a function.

Key Words

  • Decorator
  • Unpacking operator

Notes

  • Note that a decorator is a higher order function because it accepts another function as an argument.
  • In Python, functions are first-class objects:
    • Treated like any other object
    • Have types
    • Can be used in expressions
    • Can be passed to or return by functions.
