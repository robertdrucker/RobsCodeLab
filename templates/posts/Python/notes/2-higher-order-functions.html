Summary

In this post the concepts of first-class objects, higher order programming, lambda expressions, and abstraction are explored. These concepts are applied to a Root Finding program that uses the Newton-Raphson method.

Key Words

  • first-class object
  • higher-order programming
  • lamda expression

Notes

  • In Python, functions are first-class objects. As such, they can be used just like objects of other types. They can be referenced in an expression, used as a function argument, returned by a function, etc.
  • Related to first-class objects is the concept of higher-order programming. This style of coding involves passing a function or a lambda expression as an argument into another function. The key idea here is that higher-order programming allows us to make functions more general.
