Summary

Here are notes and code behind my project Wings Photo Gallery. This project features an image gallery that was built using a CSS grid layout and a little bit of JavaScript. Additionally, media queries were incorporated into the mark-up language to make the project page responsive.

Key Words

  • grid container, column axis, row axis, grid item, grid line, gutter, track, grid area, grid cell

Notes

  • Unlike flexbox, which is one dimensional, a grid allows us to make two-dimensional layouts. That is, a grid can span items across multiple columns and multiple rows.
  • This project uses a one-dimensional grid system, so it could have been built using flexbox equally well.
  • The CSS grid and flexbox layouts essentially make float designs unnecessary and obsolete. Thank goodness for this. Float designs are generally messy, difficult to implement, and hard to read.
  • There are many grid properties, and only a few are used in this project. More advanced grid features will appear in future projects.
  • For more information about CSS grids, see the references below.
